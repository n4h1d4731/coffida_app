## Config
# Backend REST server
1. Run 'npm install' from the './backend' directory to install the dependencies
2. Create your .env file with your DB config details in './backend' directory
3. Create a directory called 'storage', within the storage directory, create a directory called 'photos'
4. Copy the contents of the yaml file in the resources directory into editor.swagger.io
5. Run the server with 'npm start' from the './backend' directory to start the server

# frontend react-native application
1. Run 'npm install' from the './frontend/Coffida' directory to install the dependencies
2. Run an instance of android via an AVD from android studio or through the Android SDK
3. Run 'npx react-native run-android' from the './frontend/Coffida' directory to run the app in the android device
